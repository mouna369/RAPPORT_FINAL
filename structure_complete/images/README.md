# Dossier Images

Placez vos images ici :

## Images requises :

1. **logo_fac.jpg** - Logo de la faculté
2. **Logo_Algérie_Télécom.svg** - Logo d'Algérie Télécom

## Format recommandé :

- Format : JPG, PNG, SVG
- Résolution : 300 DPI minimum pour l'impression
- Taille : Optimisée pour le web et l'impression

## Utilisation dans les chapitres :

```typst
// Depuis un chapitre
#image("../images/logo_fac.jpg", width: 2.5cm)

// Depuis le fichier principal
#image("images/logo_fac.jpg", width: 2.5cm)
```
